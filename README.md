# About
pyscrape is a modular toolkit for automatic scraping

# Usage
```
[-h/--help]         Displays this help menu
    Examples:
      main.py <-h/--help>
      main.py <-h/--help> [arguments]
      main.py <-h/--help> execute
      main.py <-h/--help> execute help

[-e/--execute]      Executes scripts provided as parameters
    Examples:
      main.py <-e/--execute> [scripts]
      main.py <-e/--execute> google
      main.py <-e/--execute> google duckduckgo

[-l/--list]         Lists all available scripts
    Examples:
      main.py <-l/--list>
```
